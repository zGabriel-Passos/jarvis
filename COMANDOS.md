# 🤖 Guia de Comandos — Jarvis com IA

O Jarvis agora usa **Inteligência Artificial** para entender seus comandos. **Não precisa mais falar de um jeito específico!** Fale naturalmente que a IA interpreta e executa.

## Como Funciona

```
Você fala → IA entende → IA executa + responde em voz
```

## 🌐 Abrir Aplicativos

Fale o nome do app que quiser abrir. Exemplos:

- "Abre o Roblox pra mim"
- "Abre o WhatsApp"
- "Abre o Spotify"
- "Abre o Discord"
- "Abre o Steam"
- "Abre o navegador"
- "Open Edge"
- "Abre o Notepad"

## ⌨️ Comandos do Navegador

- "Nova aba" → Abre nova aba (Ctrl+T)
- "Fecha a aba" → Fecha aba atual (Ctrl+W)
- "Atualiza a página" → Recarrega (F5)
- "Volta" → Página anterior (Alt+←)
- "Avança" → Próxima página (Alt+→)

## 📋 Editar e Clipboard

- "Copia" → Copia seleção (Ctrl+C)
- "Cola" → Cola (Ctrl+V)
- "Corta" → Recorta (Ctrl+X)
- "Salva" → Salva arquivo (Ctrl+S)
- "Desfaz" → Ctrl+Z
- "Seleciona tudo" → Ctrl+A

## 🪟 Janelas

- "Minimiza essa janela" → Win+↓
- "Maximiza" → Win+↑
- "Fecha a janela" → Alt+F4
- "Minimiza tudo" → Win+D

## ✍️ Digitar Texto

- "Escreva [qualquer coisa]" → A IA digita o texto
- "Write hello world" → Digita em inglês

## 🔑 Pressionar Teclas

- "Pressiona Enter"
- "Pressiona F11" (tela cheia)
- "Pressiona Tab"
- "Pressiona Escape"
- "Pressiona Espaço"

## 💬 Conversar com a IA

Pergunte qualquer coisa! A IA responde em português:

- "Oi, como você tá?" → "Oi! Estou ótimo, pronto para ajudar!"
- "Quem é você?" → "Sou o Jarvis, seu assistente de voz..."
- "Que horas são?" → A IA responde (se souber o contexto)

## 💡 Dicas

1. **Fale naturalmente** — Não precisa memorizar comandos
2. **A IA entende variações** — "Abre o Edge", "Pode abrir o Edge", "Quero abrir o navegador" funcionam
3. **A IA confirma em voz** — Você sempre vai ouvir o que foi feito
4. **Se errar** — Repita de outro jeito, a IA é flexível
